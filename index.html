Política de Privacidade — Top Tipster World Última atualização: 31/10/2025 O aplicativo Top Tipster World respeita sua privacidade. Não coletamos, armazenamos ou compartilhamos dados pessoais dos usuários. 1. Informações coletadas O aplicativo não coleta informações de identificação pessoal (como nome, e-mail ou localização). O único dado processado é o token de notificação fornecido pelo Firebase Cloud Messaging (FCM), usado exclusivamente para enviar alertas de novas dicas de apostas. 2. Como usamos as informações O token do FCM é usado apenas para envio de notificações. Nenhum dado é compartilhado com terceiros, exceto o serviço do Firebase, que cumpre as políticas de privacidade do Google. 3. Armazenamento As informações são temporárias e podem ser excluídas a qualquer momento pelo usuário ao remover o aplicativo. 4. Links externos O aplicativo pode abrir páginas externas (como Bet365, Betano, etc.). Esses sites têm suas próprias políticas de privacidade; recomendamos que o usuário as leia antes de interagir. 5. Responsabilidade O Top Tipster World não realiza apostas, não gerencia fundos e não tem vínculo com casas de apostas. Todas as ações são realizadas diretamente pelo usuário em sites externos. 6.
